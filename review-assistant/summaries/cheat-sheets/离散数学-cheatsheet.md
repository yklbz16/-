# 离散数学 考前速记单

## 一、 命题逻辑
### 1. 核心等价公式 (德摩根律与蕴含)
- ¬(P∨Q) ⇔ **¬P∧¬Q**
- ¬(P∧Q) ⇔ **¬P∨¬Q**
- P→Q ⇔ **¬P∨Q**
- P↔Q ⇔ **(P→Q)∧(Q→P)**

### 2. 范式化简 (高频易错点)
- **主析取范式 (PDNF)**：析取 (∨) 的**小项 (mᵢ)** 之和。小项是使公式为真的所有变量组合的合取。
  - *例：P→Q ⇔ m₀∨m₂∨m₃*
- **主合取范式 (PCNF)**：合取 (∧) 的**大项 (Mᵢ)** 之积。大项是使公式为假的所有变量组合的析取。
  - **易错点**：混淆小项和大项的下标与真值指派对应关系。

### 3. 推理规则
- **附加前提法 (CP)**：要证 H₁∧H₂⇒C，可等价证 H₁⇒(H₂→C)。
- **量词推理**：
  - **US (全称例化)**：(∀x)P(x) ⇒ P(c)
  - **UG (全称推广)**：P(c) ⇒ (∀x)P(x) (**c是任意的**)
  - **ES (存在例化)**：(∃x)P(x) ⇒ P(c) (**c是特定的**)
  - **EG (存在推广)**：P(c) ⇒ (∃x)P(x)

## 二、 谓词逻辑
### 1. 命题符号化 (特性谓词)
- “所有A都是B”：**(∀x)(A(x)→B(x))**
- “存在A是B”：**(∃x)(A(x)∧B(x))**
  - **易错点**：全称用“→”，存在用“∧”，不可混淆。

### 2. 关键等价与推理
- **量词否定**：
  - ¬(∀x)P(x) ⇔ **(∃x)¬P(x)**
  - ¬(∃x)P(x) ⇔ **(∀x)¬P(x)**
- **量词辖域扩张** (前提：B不含x)：
  - (∀x)P(x)∨B ⇔ **(∀x)(P(x)∨B)**
  - (∃x)P(x)∧B ⇔ **(∃x)(P(x)∧B)**

### 3. 前束范式 & 换名/代入
- **前束范式**：所有量词提到公式最左边。使用上述等价式逐步提出。
- **换名规则**：改变**约束变元**的名称，避免与其它变元冲突。
  - 例：(∀x)(P(x)→R(x,y))∧Q(x,y) → **(∀z)(P(z)→R(z,y))∧Q(x,y)**
- **代入规则**：替换**自由变元**，替换项不得包含原公式中的约束变元。
  - 例：(∃x)(P(y)∧R(x,y)) → **(∃x)(P(z)∧R(x,z))**

## 三、 集合与关系
### 1. 关系性质与闭包 (高频考点)
| 闭包类型 | 定义 (最小扩展) |
| :--- | :--- |
| 自反闭包 r(R) | R ∪ **{<x,x> | x∈X}** |
| 对称闭包 s(R) | R ∪ **R⁻¹** (逆关系) |
| 传递闭包 t(R) | **Warshall算法**或 R ∪ R² ∪ R³ ∪ ... |

- **Warshall算法要点**：逐列扫描关系矩阵，若`a_ji=1`，则将第`i`行逻辑加到第`j`行。

### 2. 等价关系与相容关系
- **等价关系** = **自反 + 对称 + 传递**。
  - 等价类划分集合。
- **相容关系** = **自反 + 对称** (不一定传递)。
  - **最大相容类**：极大的完全子图。
  - **完全覆盖**：所有最大相容类的集合。

## 四、 函数与代数结构
### 1. 函数性质判断
- 判断 f∪g, f∩g 等是否仍为函数，关键是检查**单值性**（每个自变量是否只对应一个因变量）。
  - **易错点**：f∩g可能不再是函数（若某些自变量对应不同值）。

### 2. 代数系统对比
| 结构 | 定义 |
| :--- | :--- |
| **半群** | 封闭 + **结合律** |
| **独异点** | 半群 + **有单位元** |
| **群** | 独异点 + **每个元素有逆元** |

### 3. 群的关键性质
- 消去律成立。
- 方程 `a*x=b` 和 `y*a=b` 有唯一解。
- **循环群**：由某个元素反复运算可生成整个群，该元素为**生成元**。

## 五、 图论
### 1. 树的性质 (设|V|=n, |E|=e)
- **连通无回路图** ⇔ 树。
- **e = n - 1**。
- 若有`t`个树叶，其余结点度数≥2，则 **∑deg(v) = 2(n-1)**。
  - *例题公式*：`t + ∑(i≥2) i * nᵢ = 2(n-1)`，其中`nᵢ`是度为`i`的结点数。

### 2. 最优二叉树 (Huffman算法)
- 步骤：1. 将权值排序；2. 取**最小两个**合并为新结点；3. 重复至只剩一个根。
- 用于构造**前缀码**，实现最短编码。

### 3. 邻接矩阵
- `A`的`(i,j)`元为1表示有从`vᵢ`到`vⱼ`的边。
- `Aᵏ`的`(i,j)`元表示从`vᵢ`到`vⱼ`**长度为k**的路径数。
